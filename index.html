الذكاء الاصطناعي

& Statistics with Python & R

برنامج عملي يُعلّم علماء البيانات والإحصائيين كيفية استخدام أدوات الذكاء الاصطناعي الحديثة — النماذج اللغوية الكبيرة، هندسة الأوامر، والتحليل بمساعدة الذكاء الاصطناعي — لإنجاز أبحاث أفضل وأسرع. A practical program that teaches data scientists and statisticians how to use modern AI tools — large language models, prompt engineering, RAG, and AI-assisted analysis — to do better research, faster.
